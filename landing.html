完整响应式设计，手机浏览器即可游玩。嫌疑人面板自动收起为底部抽屉，触控操作流畅。随时随地开一局。

使用指南
三步开玩
01

启动服务器

双击 启动游戏.command 或运行 ./start.sh，服务器在 localhost:8080 启动。

02

创建房间

打开浏览器，输入昵称，点击「启幕 · 开启新局」。把 6 位房间号发给朋友。

03

开始游戏

房主选择剧本 → 分配角色 → 按流程推进：自我介绍 → 搜证 → 讨论 → 投票 → 揭晓真相。

快速开始
开发者接入
需要 Node 20+ 和 pnpm。
# 克隆仓库
git clone <repo-url>
cd murder-mystery-game

# 安装依赖
pnpm install

# 启动（构建 + 运行）
./start.sh
→ http://localhost:8080

# 或开发模式（跳过构建）
./start.sh --dev

# 一键生产新剧本
pnpm produce --players 6 --theme "校园密室"

# 全量测试
pnpm -r test # 56 tests, 0 failures
pnpm typecheck
