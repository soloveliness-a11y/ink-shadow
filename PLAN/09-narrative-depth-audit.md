## 一、角色行为逻辑链逐人审计

### 管家 赵安（凶手）

| 维度 | 内容 | 评分 |
|------|------|------|
| 动机 | 三天前发现新遗嘱取消养老金+解雇信。三十年被否定。 | ★★★★★ |
| 预备 | 藏铁丝→偷砒霜（21:00）→书房医书做功课 | ★★★★★ |
| 执行 | 泡参茶下毒（22:00）→端茶入书房（22:15）→退出等死 | ★★★★★ |
| 收尾 | 听尸→丢铁丝进花园花圃（22:25）→发现尸体（22:30）→扶正茶杯 | ★★★★★ |
| 破绽 | 茶提前一刻钟、停留超时、领口歪了、花园鞋印+花肥、工具棚撬痕 | ★★★★☆ |

**裁定**：自洽且精巧。唯一待解释的行为——**为何提前一刻钟送茶**——在「错失真凶」结局中被完美闭环：夫人下午把钟拨快了15分钟（为了准时出席晚宴），管家按快钟时间送茶。这个细节是天才级的伏笔回收。但问题是：**普通玩家在投票前完全无法获得这个信息**，导致「提前一刻钟」在他们看来只是一个无法解释的破绽——而非指向凶手的推理线索。

**建议**：在某个场景或线索中埋下「钟快了」的暗示。比如 cl_banquet_menu 的座次表备注里加一句「夫人比请柬时间早到十五分钟」——玩家需要反推钟可能快了。

## 二、信息揭示节奏逐阶段审计

```
选角 → 发本 → backstory → prologue → 自我介绍 → 搜证1 → 讨论1 → guided → 搜证2 → 讨论2 → narrow → 投票 → 结局
```

### 节奏曲线

| 阶段 | 信息密度 | 玩家状态 | 评价 |
|------|---------|---------|------|
| 选角 | 低 | 好奇，选择角色 | ✅ |
| 发本 | **极高** | 信息爆炸，消化私人剧本 | ⚠️ 一次性塞入过多信息 |
| backstory | 高 | 理解角色背景 | ✅ |
| prologue | 高 | 开始关注他人行为 | ✅ |
| 自我介绍 | 中（取决于玩家） | 社会推理，试探 | ✅ |
| 搜证1 | 中高 | 发现物证，开始拼图 | ✅ |
| 讨论1 | 中 | 分享+辩论 | ✅ |
| guided | 中 | **第一次关键时刻**：夫人和苏蔓独立发现管家提前一刻钟 | ⚠️ 靠玩家分享 |
| 搜证2 | 高 | 核心指证线索（管家破绽、鞋印、铁丝） | ✅ |
| 讨论2 | 中高 | 深入推理 | ✅ |
| narrow | 低 | 锁定嫌疑人 | ⚠️ 太晚了 |
| 投票 | 极低 | 选择 | ✅ |

### 三个审计发现

#### 发现 #1：「提前一刻钟」是关键但脆弱的线索

**问题**：管家提前一刻钟端茶是整个剧本中最微妙的破绽。它被夫人和苏蔓分别在 guided 中独立发现——这设计很好（双人独立验证）。但**这条信息的暴露完全依赖于这两个角色在讨论中主动说出来**。

如果夫人和苏蔓都不分享这个观察，那么：
- 玩家只知道「管家送了参茶」（cl_teacup）
- 不知道「时间提前了」
- 缺少一个关键的异常行为标记

**严重性**：中等。不是逻辑断裂，但存在「最好的线索永远不被发现」的风险。

**建议**：在 cl_teacup 的线索描述中增加一个细节「管家平时十点半送茶，厨娘记得今晚他十点就开始备茶了」。这样即使夫人和苏蔓不分享，搜证也能发现时间异常。

#### 发现 #2：p_guided 的反思内容与讨论脱节

**问题**：p_guided 的内容是角色在「第一轮讨论后」的个人反思。但玩家在 p_guided 阶段看到的是自己内心的独白——不一定会在讨论中说出来。如果玩家说了「提前一刻钟」的观察，那 p_guided 就是自然的。如果玩家没说，p_guided 就变成了「只有我一个人知道但没人会知道我知道」的信息孤岛。

**建议**：在 p_guided 的 narrative 前加一句引导文本，如「第一轮讨论结束，你回到房间，把听到的消息和自己的想法梳理了一遍。以下是你此刻的判断——你可以选择在第二轮讨论中分享，也可以留作底牌。」

#### 发现 #3：投票前缺少信息汇总

**问题**：从搜证2→讨论2→narrow→投票，信息是碎片化的。每个玩家收集了不同的线索，做了不同的推理，但**没有一个环节把所有已被公开的线索拼在一起**。

**影响**：投票可能变成「凭感觉」而非「凭推理」。特别是对于非推理型玩家，两轮搜证+两轮讨论积累的信息量可能已经溢出。

**建议**：在 narrow 阶段开始时，系统自动生成一份「已公开线索清单」和「关键事实汇总」展示给所有玩家。不给结论，只帮整理。

## 三、误导设计质量评估

| 角色 | 误导强度 | 误导逻辑 | 逻辑质量 | 烟雾弹等级 |
|------|---------|---------|---------|-----------|
| **顾清** | ★★★★★ | 明早对账=今晚动机最强。改账+听见闷响不开门=行为可疑。但她没杀。 | ★★★★★ | 最佳烟雾弹 |
| **苏蔓** | ★★★★☆ | 当场争执+扇巴掌。被抛弃的情妇。但她怀孕+手疼=下毒可行性低。 | ★★★★☆ | 强烟雾弹但有合理排除路径 |
| **侄儿** | ★★★☆☆ | 赌债+伪造签名+被当众羞辱。但睡着了=半不在场证明。 | ★★★☆☆ | 中等烟雾弹，易排除 |
| **夫人** | ★★☆☆☆ | 婚姻不幸+旧遗嘱。但未动筷=更像受害者。 | ★★★☆☆ | 弱烟雾弹 |
| **医生** | ★☆☆☆☆ | 掌管药柜=唯一毒源。锁被撬=不是他。 | ★★★☆☆ | 几乎可排除 |

### 误导链的「最佳路径」

如果玩家走最佳推理路径：

1. 搜证1发现新遗嘱 → 管家和侄子动机上升
2. 搜证1发现毒茶杯 → 确认砒霜，参茶=管家
3. 搜证1发现门框划痕 → 铁丝=管家
4. 搜证1发现翡翠碎片 → 苏蔓确实争执了，但不等于下毒
5. 讨论1：顾清的「明早对账」被分享 → 顾清动机最紧迫
6. guided：夫人和苏蔓都注意到管家端茶提前一刻钟
7. 搜证2发现管家破绽 → 行为证据
8. 搜证2发现鞋印+铁丝 → 物证
9. 搜证2发现工具棚锁 → 铁丝手法一致
10. 搜证2发现借据 + 账页 → 侄子和顾清的经济问题
11. 讨论2 → 拼图完成：管家=唯一机会+最强动机（三十年被否定）
12. narrow：医生确认管家

这条路径是自洽的，逻辑清晰的。**问题是：玩家能否在有限时间内走完这条路径。**

## 四、凶手漏洞梯度

### 按推理难度分层的漏洞体系

| 梯度 | 漏洞 | 发现方式 | 推理难度 |
|------|------|---------|---------|
| L1 | 参茶全程管家经手 | 搜证 cl_teacup | 低 |
| L1 | 新遗嘱剥夺管家一切 | 搜证 cl_will | 低 |
| L1 | 停留超时+神色慌张 | 搜证 cl_butler_alibi | 低 |
| L2 | 铁丝撬药柜+锁孔划痕 | 搜证 cl_arsenic（医生私密，需他公开）+ cl_door_scratch | 中 |
| L2 | 医书折角做功课 | 搜证 cl_poison_manual | 中 |
| L2 | 花园鞋印+花肥+铁丝 | 搜证 cl_garden_footprint | 中 |
| L2 | 工具棚铁丝撬痕一致 | 搜证 cl_shed_lock | 中 |
| L3 | 端茶提前一刻钟 | guided → 夫人/苏蔓分享 | 高 |
| L3 | 手腕多转了一下 | guided → 苏蔓分享 | 极高 |
| L3 | 钟快了（根本原因） | 仅在「错失真凶」结局揭示 | 不可能 |

### 梯度评估

L1 的三条是**基线怀疑**——任何正常玩家都应该能得出「管家有重大嫌疑」。
L2 的五条是**推理核心**——需要多线索交叉验证才能形成完整证据链。
L3 的两条是**决定性细节**——如果能被分享，管家的定罪率接近 100%。

**问题**：L3 的暴露完全依赖玩家行为。如果夫人和苏蔓都在讨论中保持沉默，管家的定罪可能跌到 L1+L2 的程度——此时嫌疑人可能扩散到顾清（她有最强的时机动机）。

**建议**：在 cl_teacup 的线索描述中植入「时间异常」的客观发现（见发现 #1 建议），将 L3 的部分信息降级到 L2 的可搜索层次。

## 五、高潮与反转评估

### 当前高潮点

| 位置 | 内容 | 强度 |
|------|------|------|
| prologue | 每个人看到「命案前夜」的交叉观察 | ★★★☆☆ |
| 搜证1后讨论 | 新遗嘱+毒茶 → 动机和手段初步浮现 | ★★★★☆ |
| 搜证2后讨论 | 管家破绽+鞋印铁丝 → 证据指向管家 | ★★★★★ |
| narrow 后 | 每人锁定嫌疑人 → 医生是唯一正确的 | ★★★★☆ |
| 投票结果 | 谁被投中 | ★★★★★ |
| 结局 | 真相揭晓+各角色命运 | ★★★★★ |

### 缺失的反转

当前剧本有两个巨大的反转潜力，但没有被充分挖掘：

#### 缺失反转 #1：顾清的「我没杀他」反转型

**现状**：顾清是全剧本动机最紧迫的人（明早对账），但她没有杀人。她的 narrow 指控知节，实际上是在指控镜子里的自己。

**潜力**：如果在讨论2中有人提出「既然顾清明早就要暴露了，她为什么不跑？」→ 顾清的回答「我跑了，三万七谁来还？」→ 这个瞬间可以成为全剧本最动容的时刻之一。目前这个转变只存在于她的内心独白中，没有机会被其他玩家看到。

**建议**：在 p_guided 的 narrative 中增加一段共享文本，暗示「有人选择留下不是因为无辜，而是因为不想逃。」

#### 缺失反转 #2：管家的「三十年」临场爆发

**现状**：管家的 narrow 指控侄子，是一段冷静的谎言。如果他被投票中，结局中他会「整理袖口」然后被带走。

**潜力**：如果管家在投票阶段被指认时，能触发一个「临场反应」——不是直接认罪，而是说一段关于「三十年」的话。这段话说得越平静越惊人：「泡了三十年参茶。七十五度。七圈。每一碗都是一样的。包括今晚。」——说完才认罪。

**建议**：在投票结果揭晓后、结局开始前，插入一段「凶手自白」环节（仅当投票正确时触发）。
