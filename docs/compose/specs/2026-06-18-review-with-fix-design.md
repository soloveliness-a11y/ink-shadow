# 校验+修复脚本设计文档

## 问题描述
当前 `review_with_mimo.py` 脚本只输出差异清单，没有自动修复功能。需要手动修复每个差异，效率低下。

## 设计目标
1. 自动解析差异格式
2. 自动修复差异
3. 修复后自动验证
4. 生成详细报告

## 差异格式
```
[位置]原图「原文」→识别「识别文」
```

## 实现方案

### 1. 差异解析
- 解析差异格式，提取原文、识别文、位置信息
- 支持多种格式变体（如缺少位置信息）

### 2. 自动修复
- 在 md 文件中查找原文
- 替换为识别文
- 记录修复位置和结果

### 3. 修复后验证
- 重新运行 mimo 校验
- 确认修复成功
- 记录验证结果

### 4. 报告生成
- 生成详细报告，包括：
  - 总页数、一致数、差异数
  - 每个差异的修复状态
  - 修复失败的原因

### 5. 命令行参数
- `--fix`：启用自动修复
- `--verify`：修复后验证
- `--report`：生成详细报告

## 文件结构
```
scripts/
├── review_with_mimo.py          # 原有校验脚本
└── review_with_fix.py           # 新增校验+修复脚本
```

## 使用方法
```bash
# 只校验
python3 scripts/review_with_fix.py "豪门系列/11-水袖情（6人）"

# 校验+修复
python3 scripts/review_with_fix.py "豪门系列/11-水袖情（6人）" --fix

# 校验+修复+验证
python3 scripts/review_with_fix.py "豪门系列/11-水袖情（6人）" --fix --verify

# 生成详细报告
python3 scripts/review_with_fix.py "豪门系列/11-水袖情（6人）" --report
```
